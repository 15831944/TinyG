README:

This is a start to create a GUI program to configure TinyG boards as well as send gcode files
to them.  This is currently NOT WORKING or DOING ANYTHING USEFUL.  This is a place for me to 
put my dev files.

However, if you insist on testing it out you will need the wxPython framework for your OS.
You can get that here:

wxPython URL:
http://www.wxpython.org/download.php#binaries
